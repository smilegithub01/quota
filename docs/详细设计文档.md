## 2. 额度生命周期管控模块详细设计

额度生命周期管控模块是系统核心模块，负责额度从创建到终止的完整生命周期管理。

### 2.1 业务流程设计

#### 2.1.1 额度创建流程

```
开始
  ↓
接收授信审批结果
  ↓
参数校验（批复编号、客户ID、额度金额等）
  ↓
合规性校验
  ├─ 单户额度 ≤ 净资本10%？
  ├─ 集团额度 ≤ 净资本15%？
  └─ 关联方额度 ≤ 净资本20%？
  ↓
校验通过？
  ├─ 否 → 返回合规校验失败
  └─ 是 → 层级约束校验
           ↓
           客户额度 ≤ 集团细项限额分配额？
           批复额度 ≤ 客户细项额度？
           ↓
           校验通过？
           ├─ 否 → 返回层级约束失败
           └─ 是 → 生成额度编号（QT+年月日+6位序号）
                    ↓
                    初始化额度数据
                    ├─ 状态：待启用
                    ├─ 已用额度：0
                    └─ 可用额度 = 总额度
                    ↓
                    建立层级关联关系
                    ↓
                    记录创建日志
                    ↓
                    返回创建成功
结束
```

#### 2.1.2 额度调整流程

```
开始
  ↓
接收调整申请
  ↓
参数校验（额度ID、调整类型、调整金额等）
  ↓
查询额度当前状态
  ↓
状态校验（必须为正常或冻结状态）
  ↓
调整类型判断
  ├─ 临时调整 → 校验有效期（最长1年）
  ├─ 永久调整 → 校验调整后额度 ≥ 已用额度
  └─ 批量调整 → 校验批量规则
  ↓
校验通过？
  ├─ 否 → 返回校验失败
  └─ 是 → 判断是否需要审批
           ↓
           需要审批？
           ├─ 是 → 发起审批流程
           │        ↓
           │        审批通过？
           │        ├─ 否 → 返回审批拒绝
           │        └─ 是 → 执行调整
           └─ 否 → 直接执行调整
                    ↓
                    更新额度数据
                    ├─ 总额度 = 原总额度 + 调整金额
                    ├─ 可用额度 = 原可用额度 + 调整金额
                    └─ 更新时间 = 当前时间
                    ↓
                    联动更新下层额度
                    ↓
                    记录调整历史
                    ↓
                    返回调整成功
结束
```

#### 2.1.3 额度使用（占用）流程

```
开始
  ↓
接收用信申请
  ↓
参数校验（客户ID、业务品种、用信金额、合同编号等）
  ↓
白名单校验
  ├─ 查询客户白名单状态
  └─ 判断豁免规则（完全豁免/部分豁免/阈值豁免）
  ↓
命中白名单？
  ├─ 完全豁免 → 跳过额度校验，直接创建占用记录
  └─ 否 → 执行六级递进校验
           ↓
           第一级：集团限额校验
           ├─ 集团总额度可用性
           └─ 集团细项额度可用性
           ↓
           第二级：客户额度校验
           ├─ 客户总额度可用性
           └─ 客户状态有效性
           ↓
           第三级：客户细项额度校验
           ├─ 细项额度可用性
           └─ 细项类型匹配
           ↓
           第四级：批复额度校验
           ├─ 批复额度可用性
           └─ 批复状态有效性
           ↓
           第五级：授信产品细项校验
           ├─ 产品额度可用性
           └─ 产品状态有效性
           ↓
           第六级：业务品种映射校验
           ├─ 映射关系存在性
           └─ 占用比例计算
           ↓
           任一级校验失败？
           ├─ 是 → 返回额度不足，包含缺口层级和缺口金额
           └─ 否 → 获取分布式锁
                    ↓
                    锁定成功？
                    ├─ 否 → 返回锁定失败
                    └─ 是 → 执行额度占用
                             ├─ 扣减授信产品细项额度
                             ├─ 扣减批复额度
                             ├─ 扣减客户细项额度
                             ├─ 扣减客户总额度
                             ├─ 扣减集团细项限额
                             └─ 扣减集团总限额
                             ↓
                             创建合同占用记录
                             ↓
                             记录使用明细
                             ↓
                             释放分布式锁
                             ↓
                             返回占用成功
结束
```

#### 2.1.4 额度冻结流程

```
开始
  ↓
接收冻结申请
  ↓
参数校验（额度ID、冻结类型、冻结原因等）
  ↓
查询额度当前状态
  ↓
状态校验（必须为正常状态）
  ↓
冻结类型判断
  ├─ 全额冻结 → 冻结金额 = 可用额度
  ├─ 部分冻结 → 校验冻结金额 ≤ 可用额度
  └─ 条件冻结 → 校验冻结条件
  ↓
校验通过？
  ├─ 否 → 返回校验失败
  └─ 是 → 执行冻结操作
           ├─ 更新冻结额度
           ├─ 更新可用额度
           └─ 更新状态为冻结
           ↓
           联动冻结下层额度
           ↓
           记录冻结日志
           ↓
           返回冻结成功
结束
```

#### 2.1.5 额度解冻流程

```
开始
  ↓
接收解冻申请
  ↓
参数校验（冻结记录ID、解冻原因等）
  ↓
查询冻结记录
  ↓
校验冻结记录有效性
  ↓
解冻条件校验
  ├─ 冻结原因已消除？
  └─ 相关风险已解除？
  ↓
校验通过？
  ├─ 否 → 返回校验失败
  └─ 是 → 判断是否需要审批
           ↓
           需要审批？
           ├─ 是 → 发起审批流程
           │        ↓
           │        审批通过？
           │        ├─ 否 → 返回审批拒绝
           │        └─ 是 → 执行解冻
           └─ 否 → 直接执行解冻
                    ↓
                    更新额度数据
                    ├─ 冻结额度 = 原冻结额度 - 解冻金额
                    ├─ 可用额度 = 原可用额度 + 解冻金额
                    └─ 状态 = 正常（冻结额度为0时）
                    ↓
                    联动解冻下层额度
                    ↓
                    记录解冻日志
                    ↓
                    返回解冻成功
结束
```

#### 2.1.6 额度释放流程

```
开始
  ↓
接收还款/终止事件
  ↓
参数校验（合同编号、释放金额、释放原因等）
  ↓
查询合同占用记录
  ↓
计算释放金额
  ├─ 正常还款 → 释放金额 = 还款本金
  ├─ 提前还款 → 释放金额 = 提前还款本金
  ├─ 合同终止 → 释放金额 = 剩余占用金额
  └─ 逾期核销 → 释放金额 = 核销本金
  ↓
校验释放金额 ≤ 已占用金额
  ↓
校验通过？
  ├─ 否 → 返回校验失败
  └─ 是 → 获取分布式锁
           ↓
           锁定成功？
           ├─ 否 → 返回锁定失败
           └─ 是 → 执行额度释放
                    ├─ 返还授信产品细项额度
                    ├─ 返还批复额度
                    ├─ 返还客户细项额度
                    ├─ 返还客户总额度
                    ├─ 返还集团细项限额
                    └─ 返还集团总限额
                    ↓
                    更新合同占用记录
                    ↓
                    记录释放明细
                    ↓
                    释放分布式锁
                    ↓
                    返回释放成功
结束
```

#### 2.1.7 额度终止流程

```
开始
  ↓
接收终止事件
  ↓
参数校验（额度ID、终止类型、终止原因等）
  ↓
终止前检查
  ├─ 是否有未完成的占用？
  ├─ 是否有未处理的冻结？
  └─ 是否有未完成的调整？
  ↓
检查通过？
  ├─ 否 → 返回检查失败，提示需先处理的事项
  └─ 是 → 执行终止操作
           ├─ 更新状态为已终止
           ├─ 清理相关数据
           └─ 归档历史数据
           ↓
           联动终止下层额度
           ↓
           记录终止日志
           ↓
           返回终止成功
结束
```

#### 2.3.2 DTO模型

```java
package com.bank.quota.dto.lifecycle;

import lombok.Data;
import javax.validation.constraints.*;
import java.math.BigDecimal;
import java.time.LocalDateTime;
import java.util.List;

@Data
public class QuotaCreateRequest {
    @NotBlank(message = "授信批复编号不能为空")
    private String approvalNo;
    
    @NotNull(message = "客户ID不能为空")
    private Long customerId;
    
    @NotBlank(message = "额度类型不能为空")
    private String quotaType;
    
    @NotNull(message = "额度金额不能为空")
    @Positive(message = "额度金额必须大于0")
    private BigDecimal quotaAmount;
    
    private String currency = "CNY";
```

```java
package com.bank.whitelist.strategy.impl;

import com.bank.whitelist.strategy.*;
import com.bank.whitelist.domain.WhitelistExemptItem;
import com.bank.quota.context.QuotaCheckContext;
import org.springframework.stereotype.Component;
import java.math.BigDecimal;

@Component
public class FullExemptStrategy implements ExemptStrategy {
    
    @Override
    public ExemptType getExemptType() {
        return ExemptType.FULL;
    }
    
    @Override
    public ExemptResult apply(WhitelistExemptItem exemptItem, QuotaCheckContext context) {
        BigDecimal originalAmount = context.getRequestAmount();
        return ExemptResult.fullExempt(exemptItem.getCheckPoint(), originalAmount);
    }
    
    @Override
    public boolean supports(ExemptType exemptType) {
        return ExemptType.FULL == exemptType;
    }
}

@Component
public class PartialExemptStrategy implements ExemptStrategy {
    
    @Override
    public ExemptType getExemptType() {
        return ExemptType.PARTIAL;
    }
    
    @Override
    public ExemptResult apply(WhitelistExemptItem exemptItem, QuotaCheckContext context) {
        BigDecimal originalAmount = context.getRequestAmount();
        BigDecimal exemptAmount = exemptItem.getExemptValue();
        
        if (exemptItem.getExemptRatio() != null) {
            exemptAmount = originalAmount.multiply(exemptItem.getExemptRatio());
        }
        
        if (exemptAmount.compareTo(originalAmount) > 0) {
            exemptAmount = originalAmount;
        }
        
        return ExemptResult.partialExempt(exemptItem.getCheckPoint(), 
            originalAmount, exemptAmount);
    }
    
    @Override
    public boolean supports(ExemptType exemptType) {
        return ExemptType.PARTIAL == exemptType;
    }
}

@Component
public class ThresholdExemptStrategy implements ExemptStrategy {
    
    @Override
    public ExemptType getExemptType() {
        return ExemptType.THRESHOLD;
    }
    
    @Override
    public ExemptResult apply(WhitelistExemptItem exemptItem, QuotaCheckContext context) {
        BigDecimal newThreshold = exemptItem.getOverrideThreshold();
        return ExemptResult.thresholdExempt(exemptItem.getCheckPoint(), newThreshold);
    }
    
    @Override
    public boolean supports(ExemptType exemptType) {
        return ExemptType.THRESHOLD == exemptType;
    }
}

@Component
@RequiredArgsConstructor
public class ConditionalExemptStrategy implements ExemptStrategy {
    
    private final SpelExpressionParser expressionParser = new SpelExpressionParser();
    
    @Override
    public ExemptType getExemptType() {
        return ExemptType.CONDITIONAL;
    }
    
    @Override
    public ExemptResult apply(WhitelistExemptItem exemptItem, QuotaCheckContext context) {
        String condition = exemptItem.getExemptCondition();
        
        StandardEvaluationContext evalContext = new StandardEvaluationContext();
        evalContext.setVariable("context", context);
        evalContext.setVariable("amount", context.getRequestAmount());
        evalContext.setVariable("customer", context.getCustomer());
        evalContext.setVariable("businessType", context.getBusinessType());
        
        Expression expression = expressionParser.parseExpression(condition);
        Boolean result = expression.getValue(evalContext, Boolean.class);
        
        if (Boolean.TRUE.equals(result)) {
            return ExemptResult.fullExempt(exemptItem.getCheckPoint(), 
                context.getRequestAmount());
        }
        
        return ExemptResult.notExempted(exemptItem.getCheckPoint());
    }
    
    @Override
    public boolean supports(ExemptType exemptType) {
        return ExemptType.CONDITIONAL == exemptType;
    }
```

```java
package com.bank.whitelist.strategy;

import org.springframework.stereotype.Component;
import lombok.RequiredArgsConstructor;
import java.util.List;
import java.util.Map;
import java.util.function.Function;
import java.util.stream.Collectors;
import javax.annotation.PostConstruct;

@Component
@RequiredArgsConstructor
public class ExemptStrategyFactory {
    
    private final List<ExemptStrategy> strategies;
    private Map<ExemptType, ExemptStrategy> strategyMap;
```

```java
    protected abstract CheckResult doCheckWithThreshold(QuotaCheckContext context, 
        BigDecimal newThreshold);

        }
        
        log.info("校验点 {} 命中白名单豁免，白名单编号: {}, 豁免类型: {}", 
            checkPoint.getName(), exemptItem.getWhitelistNo(), exemptItem.getExemptType());
        
        ExemptStrategy strategy = strategyFactory.getStrategy(exemptItem.getExemptType());
        ExemptResult exemptResult = strategy.apply(exemptItem, context);
        
        context.addExemptResult(checkPoint.getCode(), exemptResult);
        
        exemptService.recordUsage(exemptItem, context, exemptResult);
        
        if (exemptResult.isExempted()) {
            switch (exemptItem.getExemptType()) {
                case FULL:
                    log.info("校验点 {} 完全豁免，跳过校验", checkPoint.getName());
                    return CheckResult.exempted(checkPoint, exemptResult);
                    
                case PARTIAL:
                    log.info("校验点 {} 部分豁免，豁免金额: {}, 剩余金额参与校验", 
                        checkPoint.getName(), exemptResult.getExemptAmount());
                    context.setRequestAmount(exemptResult.getActualAmount());
                    return doCheck(context);
                    
                case THRESHOLD:
                    log.info("校验点 {} 阈值豁免，使用新阈值: {}", 
                        checkPoint.getName(), exemptResult.getNewThreshold());
                    return doCheckWithThreshold(context, exemptResult.getNewThreshold());
                    
                case CONDITIONAL:
                    if (exemptResult.isExempted()) {
                        log.info("校验点 {} 条件豁免生效，跳过校验", checkPoint.getName());
                        return CheckResult.exempted(checkPoint, exemptResult);
                    }
                    return doCheck(context);
                    
                default:
                    return doCheck(context);
            }
        }
        
        return doCheck(context);
    }
}
```

```java
@Component
public class GroupTotalLimitChecker extends AbstractQuotaChecker {
    
    private final GroupQuotaService groupQuotaService;
    
    public GroupTotalLimitChecker(WhitelistExemptService exemptService,
                                   ExemptStrategyFactory strategyFactory,
                                   GroupQuotaService groupQuotaService) {
        super(exemptService, strategyFactory);
        this.groupQuotaService = groupQuotaService;

    protected CheckPoint getCheckPoint() {

        BigDecimal requestAmount = context.getRequestAmount();

            return CheckResult.failed(getCheckPoint(), "集团额度不存在");
        }
        
        BigDecimal availableQuota = groupQuota.getAvailableQuota();
        
        if (requestAmount.compareTo(availableQuota) > 0) {
            return CheckResult.failed(getCheckPoint(), 
                String.format("集团总额度不足，可用: %s，申请: %s，缺口: %s",
                    availableQuota, requestAmount, requestAmount.subtract(availableQuota)));
        }
        
        return CheckResult.success(getCheckPoint());
    }
    
    @Override
    protected CheckResult doCheckWithThreshold(QuotaCheckContext context, 
                                                BigDecimal newThreshold) {
        Long groupId = context.getGroupId();
        BigDecimal requestAmount = context.getRequestAmount();
        
        GroupQuota groupQuota = groupQuotaService.getGroupQuota(groupId);
        
        BigDecimal effectiveAvailable = newThreshold
            .subtract(groupQuota.getUsedQuota())
            .subtract(groupQuota.getFrozenQuota());
        
        if (requestAmount.compareTo(effectiveAvailable) > 0) {
            return CheckResult.failed(getCheckPoint(),
                String.format("集团总额度不足（使用豁免阈值），可用: %s，申请: %s",
                    effectiveAvailable, requestAmount));
        }
        
        return CheckResult.success(getCheckPoint());
    }
```

```java
        WhitelistUsageLog log = WhitelistUsageLog.builder()
            .logId(generateLogId())
            .whitelistNo(exemptItem.getWhitelistNo())
            .customerId(context.getCustomerId())
            .businessType(context.getBusinessType())
            .checkPoint(exemptItem.getCheckPoint())
            .exemptType(exemptItem.getExemptType().name())
            .originalAmount(exemptResult.getOriginalAmount())
            .exemptAmount(exemptResult.getExemptAmount())
            .actualAmount(exemptResult.getActualAmount())
            .operationTime(LocalDateTime.now())
            .operator(context.getOperator())
            .ipAddress(context.getIpAddress())
            .build();
        
        usageLogRepository.save(log);
    }
```

### 7.6 设计要点总结

#### 7.6.1 架构优势

| 设计模式 | 应用场景 | 解决的问题 |
|---------|---------|-----------|
| **策略模式** | 豁免策略实现 | 不同豁免类型的处理逻辑解耦，易于扩展新的豁免类型 |
| **责任链模式** | 校验处理器链 | 每个校验点独立处理，白名单逻辑统一前置，避免代码重复 |
| **模板方法模式** | AbstractQuotaChecker | 统一白名单豁免检查流程，子类只需实现具体校验逻辑 |
| **工厂模式** | ExemptStrategyFactory | 策略实例的统一管理和获取 |

#### 7.6.2 白名单豁免类型说明

| 豁免类型 | 适用场景 | 配置参数 | 示例 |
|---------|---------|---------|------|
| **FULL** | VIP客户、战略合作伙伴 | 无需额外参数 | 客户A在集团限额校验点完全豁免 |
| **PARTIAL** | 部分金额豁免 | exemptValue/exemptRatio | 客户B在风险敞口校验豁免500万 |
| **THRESHOLD** | 提高校验阈值 | overrideThreshold | 客户C的单户限额比例从10%提高到15% |
| **CONDITIONAL** | 条件性豁免 | exemptCondition(SpEL) | 当金额<100万时豁免合规校验 |

#### 8.3.2 汇率验证器设计

```java
package com.bank.quota.service.exchangerate;

import org.springframework.stereotype.Component;
import java.math.BigDecimal;
import java.math.RoundingMode;
import java.util.List;

@Component
public class ExchangeRateValidator {
    
    private static final BigDecimal VOLATILITY_THRESHOLD = new BigDecimal("0.05");
    private static final int SIGMA_MULTIPLIER = 3;
    
    public ValidationResult validateFormat(ExchangeRateDTO rateDTO) {
        ValidationResult result = new ValidationResult();
        
        if (rateDTO.getCurrencyPair() == null || rateDTO.getCurrencyPair().isEmpty()) {
            result.addError("货币对不能为空");
        }
        
        if (!isValidCurrencyCode(rateDTO.getBaseCurrency()) || 
            !isValidCurrencyCode(rateDTO.getQuoteCurrency())) {
            result.addError("货币代码不符合ISO 4217标准");
        }
        
        if (rateDTO.getRate() == null || rateDTO.getRate().compareTo(BigDecimal.ZERO) <= 0) {
            result.addError("汇率值必须大于0");
        }
        
        result.setFormatValid(!result.hasErrors());
        return result;
    }
    
    public ValidationResult validateRange(ExchangeRateDTO rateDTO, 
                                           ExchangeRate previousRate) {
        ValidationResult result = new ValidationResult();
        
        if (previousRate != null) {
            BigDecimal change = rateDTO.getRate().subtract(previousRate.getRate())
                .divide(previousRate.getRate(), 6, RoundingMode.HALF_EVEN)
                .abs();
            
            if (change.compareTo(VOLATILITY_THRESHOLD) > 0) {
                result.addWarning(String.format(
                    "汇率波动超过阈值: %.2f%% > 5%%", 
                    change.multiply(new BigDecimal("100"))));
                result.setRangeValid(false);
            } else {
                result.setRangeValid(true);
            }
        } else {
            result.setRangeValid(true);
        }
        
        return result;
    }
    
    public ValidationResult validateAnomaly(ExchangeRateDTO rateDTO, 
                                             List<ExchangeRate> historicalRates) {
        ValidationResult result = new ValidationResult();
        
        if (historicalRates != null && historicalRates.size() >= 30) {
            BigDecimal mean = calculateMean(historicalRates);
            BigDecimal stdDev = calculateStandardDeviation(historicalRates, mean);
            
            BigDecimal deviation = rateDTO.getRate().subtract(mean).abs();
            BigDecimal threshold = stdDev.multiply(new BigDecimal(SIGMA_MULTIPLIER));
            
            if (deviation.compareTo(threshold) > 0) {
                result.addWarning(String.format(
                    "检测到异常值: 偏离均值 %.4f 个标准差", 
                    deviation.divide(stdDev, 2, RoundingMode.HALF_EVEN)));
                result.setAnomalyDetected(true);
            } else {
                result.setAnomalyDetected(false);
            }
        } else {
            result.setAnomalyDetected(false);
        }
        
        return result;
    }
    
    private BigDecimal calculateMean(List<ExchangeRate> rates) {
        BigDecimal sum = rates.stream()
            .map(ExchangeRate::getRate)
            .reduce(BigDecimal.ZERO, BigDecimal::add);
        return sum.divide(new BigDecimal(rates.size()), 6, RoundingMode.HALF_EVEN);
    }
    
    private BigDecimal calculateStandardDeviation(List<ExchangeRate> rates, 
                                                    BigDecimal mean) {
        BigDecimal sumOfSquares = rates.stream()
            .map(r -> r.getRate().subtract(mean).pow(2))
            .reduce(BigDecimal.ZERO, BigDecimal::add);
        BigDecimal variance = sumOfSquares.divide(
            new BigDecimal(rates.size()), 6, RoundingMode.HALF_EVEN);
        return variance.sqrt(java.math.MathContext.DECIMAL64);
    }
}
```

#### 9.3.2 重算执行器设计

```java
package com.bank.quota.service.recalculate;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;
import java.util.concurrent.CompletableFuture;

@Slf4j
@Service
@RequiredArgsConstructor
public class RecalculateExecutor {
    
    private final QuotaRecalculator quotaRecalculator;
    private final OccupancyRecalculator occupancyRecalculator;
    private final SummaryRecalculator summaryRecalculator;
    private final RecalculateTaskRepository taskRepository;
    private final RecalculateDiffRepository diffRepository;
    private final RedissonClient redissonClient;
    
    @Async("recalculateExecutor")
    public CompletableFuture<RecalculateResult> execute(RecalculateTask task) {
        String lockKey = "recalculate:lock:" + task.getTaskId();
        RLock lock = redissonClient.getLock(lockKey);
        
        try {
            if (!lock.tryLock(10, 3600, TimeUnit.SECONDS)) {
                return CompletableFuture.completedFuture(
                    RecalculateResult.failed("获取锁失败"));
            }
            
            task.setStatus(TaskStatus.RUNNING);
            task.setStartTime(LocalDateTime.now());
            taskRepository.save(task);
            
            RecalculateContext context = buildContext(task);
            
            int batchSize = 1000;
            int processed = 0;
            
            List<QuotaData> quotaList = fetchAffectedQuotas(context);
            task.setTotalRecords(quotaList.size());
            
            for (int i = 0; i < quotaList.size(); i += batchSize) {
                List<QuotaData> batch = quotaList.subList(i, 
                    Math.min(i + batchSize, quotaList.size()));
                
                for (QuotaData quota : batch) {
                    try {
                        RecalculateResult result = recalculateQuota(quota, context);
                        
                        if (result.hasDifference()) {
                            saveDifference(task.getTaskId(), result);
                        }
                        
                        processed++;
                        task.setProcessedRecords(processed);
                        task.setProgress(calculateProgress(processed, quotaList.size()));
                        
                    } catch (Exception e) {
                        log.error("重算失败: quotaId={}", quota.getId(), e);
                        task.setFailCount(task.getFailCount() + 1);
                    }
                }
                
                taskRepository.save(task);
            }
            
            task.setStatus(TaskStatus.COMPLETED);
            task.setEndTime(LocalDateTime.now());
            taskRepository.save(task);
            
            return CompletableFuture.completedFuture(RecalculateResult.success());
            
        } catch (Exception e) {
            log.error("重算任务执行异常: taskId={}", task.getTaskId(), e);
            task.setStatus(TaskStatus.FAILED);
            task.setErrorMessage(e.getMessage());
            taskRepository.save(task);
            return CompletableFuture.completedFuture(RecalculateResult.failed(e.getMessage()));
            
        } finally {
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }
    
    private RecalculateResult recalculateQuota(QuotaData quota, RecalculateContext context) {
        BigDecimal oldAmount = quota.getAmount();
        BigDecimal newAmount = calculateNewAmount(oldAmount, context);
        
        if (oldAmount.compareTo(newAmount) == 0) {
            return RecalculateResult.noChange();
        }
        
        quota.setAmount(newAmount);
        quotaRecalculator.updateQuota(quota);
        
        return RecalculateResult.withDifference(
            quota.getId(), quota.getQuotaType(), 
            oldAmount, newAmount);
    }
}
```

```
定时任务触发（每日01:00）
  ↓
创建同步任务
  ├─ 同步类型：DAILY_BATCH
  ├─ 数据源：CORE_BUSINESS
  └─ 同步模式：INCREMENTAL
  ↓
查询上次同步检查点
  ↓
存在检查点？
  ├─ 是 → 从检查点位置继续
  └─ 否 → 从昨日00:00开始
           ↓
           数据抽取阶段
           ├─ 连接源数据库
           ├─ 执行增量查询
           │   └─ WHERE update_time > ? AND update_time <= ?
           ├─ 分批获取数据（每批5000条）
           └─ 数据格式转换
           ↓
           数据传输阶段
           ├─ 批量发送到消息队列
           ├─ 记录传输检查点
           └─ 监控传输进度
           ↓
           数据加载阶段
           ├─ 消费消息队列数据
           ├─ 执行批量插入/更新
           ├─ 处理数据冲突
           │   ├─ 存在则更新
           │   └─ 不存在则插入
           └─ 记录处理结果
           ↓
           完整性校验
           ├─ 源系统记录数 = 目标系统记录数？
           ├─ 关键字段一致性校验
           └─ 业务规则校验
           ↓
           校验通过？
           ├─ 否 → 记录差异，发送告警
           └─ 是 → 更新任务状态为完成
                    ↓
                    记录同步日志
                    ↓
                    清理临时数据
                    ↓
                    返回同步成功
结束
```
