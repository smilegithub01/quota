## 9. 数据流图 - 额度占用与释放

```mermaid
sequenceDiagram
    participant Client as 业务系统
    participant Gateway as API网关
    participant Control as 额度控制服务
    participant Query as 额度查询服务
    participant Calc as 额度计算服务
    participant Redis as Redis缓存
    participant DB as MySQL数据库

    Note over Client,DB: 额度占用流程
    Client->>Gateway: 用信申请
    Gateway->>Control: 转发请求
    Control->>Query: 校验额度充足性
    Query->>DB: 查询额度数据
    DB-->>Query: 返回额度信息
    Query-->>Control: 充足性结果
    
    alt 额度充足
        Control->>Redis: 获取分布式锁
        Redis-->>Control: 锁获取成功
        Control->>Calc: 计算占用金额
        Calc-->>Control: 返回计算结果
        Control->>DB: 扣减额度(穿透)
        DB-->>Control: 扣减成功
        Control->>DB: 创建占用记录
        Control->>Redis: 释放分布式锁
        Control-->>Gateway: 占用成功
        Gateway-->>Client: 返回成功
    else 额度不足
        Control-->>Gateway: 返回额度不足
        Gateway-->>Client: 返回失败
    end

    Note over Client,DB: 额度释放流程
    Client->>Gateway: 还款/合同终止
    Gateway->>Control: 释放请求
    Control->>DB: 查询占用记录
    DB-->>Control: 返回占用信息
    Control->>Calc: 计算释放金额
    Calc-->>Control: 返回计算结果
    Control->>DB: 释放额度(穿透)
    DB-->>Control: 释放成功
    Control->>DB: 更新占用记录
    Control-->>Gateway: 释放成功
    Gateway-->>Client: 返回成功
```
